# 植物百科网页应用 🌿

一个现代化的植物识别与养护管理平台，基于React + Ant Design开发。

## ✨ 功能特性

### 🏠 首页模块
- 现代化hero区域设计
- 功能卡片展示（搜索、识别、提醒）
- 热门植物展示区
- 响应式布局和动画效果

### 🔍 搜索模块
- 实时搜索功能
- 植物分类筛选
- 搜索结果展示
- 空状态处理

### 📷 图片识别模块
- 图片上传组件
- AI识别功能（模拟）
- 识别结果展示
- 自动识别流程

### ⏰ 养护提醒模块
- 提醒列表管理
- 添加/删除提醒
- 植物关联选择
- 提醒类型配置

### 📖 植物详情模块
- 详细信息展示
- 图片轮播
- 养护要点
- 常见问题解答

## 🛠 技术栈

- **前端框架**: React 18.2.0
- **UI组件库**: Ant Design 5.12.8
- **路由管理**: React Router DOM 6.21.1
- **构建工具**: Vite 5.0.8
- **样式方案**: CSS变量 + 自定义样式

一个基于React和Vite的现代植物识别与养护管理平台。

## 功能特性

- 🌱 **植物搜索**: 快速查找植物信息和养护知识
- 📸 **图片识别**: 上传照片识别植物种类
- ⏰ **养护提醒**: 设置个性化的植物养护计划
- 📱 **响应式设计**: 适配各种设备屏幕
- 🎨 **现代UI**: 采用Ant Design组件库，界面美观现代

## 技术栈

- **前端框架**: React 18
- **构建工具**: Vite
- **UI组件库**: Ant Design
- **路由管理**: React Router
- **样式**: CSS变量 + 现代CSS特性
- **开发环境**: Node.js 16+

## 功能模块

### 1. 首页 (HomePage)
- 搜索框：快速搜索植物
- 功能卡片：拍照识别、植物搜索、养护提醒
- 热门植物展示：展示常见植物

### 2. 搜索页面 (SearchPage)
- 关键词搜索：支持植物名称、特征搜索
- 搜索结果：卡片式展示搜索结果
- 筛选功能：按分类筛选植物

### 3. 图片识别页面 (RecognitionPage)
- 图片上传：支持拖拽和点击上传
- 拍照功能：使用摄像头拍照识别
- 识别结果：展示识别到的植物信息

### 4. 养护提醒页面 (ReminderPage)
- 提醒列表：管理所有养护提醒
- 添加提醒：为植物设置养护计划
- 编辑删除：修改或删除提醒

### 5. 植物详情页面 (PlantDetailPage)
- 基础信息：植物名称、科属、产地等
- 养护要点：详细的养护指导
- 参考图片：植物图片展示
- 养护提醒：添加个性化提醒
